# 无重复字符的最长子串
给定一个字符串，请你找出其中不含有重复字符的 **最长子串** 的长度。  

## 示例一：

``` c
输入: "abcabcbb"
输出: 3 
解释: 因为无重复字符的最长子串是 "abc"，所以其长度为 3。
```

## 示例二：

``` c
输入: "bbbbb"
输出: 1
解释: 因为无重复字符的最长子串是 "b"，所以其长度为 1。
```

## 示例三：

``` c
输入: "pwwkew"
输出: 3
解释: 因为无重复字符的最长子串是 "wke"，所以其长度为 3。  
     请注意，你的答案必须是 子串 的长度，"pwke" 是一个子序列，不是子串。
```

### 代码(滑动窗口)：

``` c
/*
设立左指针j 和 右指针 i
i 指针向右侧伸缩{
    对每个s[i] 判断是否在之前的数组出现过；
        如果出现，指针j 指向出现过的位置的下一个位置；
    更新右指针 和 最大长度；
}
时间复杂度 O(n*n)
空间复杂度O(1)
*/
int lengthOfLongestSubstring(char * s){
    int n = strlen(s);
    int i, j = 0, k;
    int max = 0;
    for(i=0; i<n; i++)
    {
        for(k=j; k<i; k++)
        {
            if(s[k] == s[i])
            {
                j = k+1;  
                break;
            }
        }
        if (max < i-j+1)
            max = i-j+1;
    }
    return max;
}
```
